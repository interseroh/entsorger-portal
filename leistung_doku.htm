Interseroh Entsorger Portal Leistungen API

Beschreibung

Interseroh Entsorger Portal Leistungen API bildet Leistungsmeldungen eines Entsorgungspartner an Interseroh ab. Ein Leistungsmeldung liefert alle relevanten Informationen nach Durchführung einer Dienstleistung. Sie ist Voraussetzung für eine Abrechnung,

Technische Dokumentation

Mehrwert

Ein System, dass den Service Interseroh Entsorger Portal Leistungen API konsumiert kann automatisiert Leistungsmeldungen an Interseroh übermitteln.

Funktionalität

Der Service Interseroh Entsorger Portal Leistungen API kann über verschiedene Endpoints genutzt werden. Diese Endpoints sind durch Swagger dokumentiert (Link zu Swagger).
Folgende Funktionalitäten werden bereitgestellt:

  • Empfang von Leistungsmeldungen in Form von Metadaten.
  • Empfang von Belegen in Form von Bilddateien.

Der gesamte Service basiert auf dem Datenobjekt Leistung. Das Objekt hat Paramter und verweist auf weitere Objekte. Diese verfügen wiederum über eigene Paramter. Im folgenden werden die Objekte und Parameter erklärt:

Objekt Parameter Erläuterung

meldung		 Leistungsmeldung
datum Leistungsdatum
masseinhalt Muss der Einheit der Bestellposition vom Typ FRACTION entsprechen.
menge Menge des Materials
verwogen Gibt an, ob das Material verwogen wurde. Mögliche Werte: TRUE; FALSE
beleg Leistungsbeleg
nummer Eindeutige Nummer des Beleges.
typ Typ des Belegs. Mögliche Werte: WIEGESCHEIN; LIEFERSCHEIN; SONSTIGES
